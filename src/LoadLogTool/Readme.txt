程序编制：李世存


Usage：

1、配置conf.ini文件里的信息

2、双击LoadLogTool.bat下载服务器中指定的日志，建议把此文件建一个快捷方式快捷栏，执行用例后双击一下即可下载


注：如果程序不能运行，请确认PC是否安装了Java,如已安装请确认是否配置了环境变量，简单的检查文件，在cmd中打入：javac 如果出现命令没有找到的提示说明环境变理没有配置.

Java环境变量配置：

JAVA_HOME
C:\jdk1.5.0_06 //根据实际情况修改

CLASSPATH
.;%JAVA_HOME%\lib

Path
%JAVA_HOME%\bin
